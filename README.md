# ComposeNotesApp
Приложение-заметки, написанное с использованием Jetpack Compose 
# Что использовалось:
- **Jetpack Compose** - Построение UI
- **MVI** - Архитектура слоя представления
- **Koin** - Dependency injection
- **Realm** - NoSQL база данных
